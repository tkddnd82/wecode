#git 명령어

##git init
git을 추적 실행하기 위한 명령어
명령어는 항상 해당 폴더에서 실행 

##git add .
수정한거 저장

##git commit -m "test"
수정한 저장사항에 대한 책갈피같은 느낌

##git remote add origin 주소
git remote add origin git@github.com:tkddnd82/wecode.git 
push 하기 전에 내가 생성한 디렉토리와 repository 주소의 연동

#git branch branch명
브렌치 생성

#git checkout branch명
해당 브렌치로 이동
